基于 ui-ux-pro-max 设计系统：移动端「AI Agent 指挥官 + 结果审阅台」。每一行是一个视觉版本，横向排列该版本下的 7 个核心页面（连接 / 首页 / 会话 / Diff 审阅 / 终端 / Web 预览联调 / 检查点历史）。横向可滚动查看。V1–V8 既有风格；V9–V13 为 brand skill 新增的 5 个品牌（Graphite Carbon / Sunrise Coral / Matrix Forest / Slate Pro / Midnight Plum），各画 P1–P5 五个页面。
